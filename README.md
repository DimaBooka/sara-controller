Home assistant SARA

To run it, just type:

```python main.py```

and say "Hey, Sara!"

---

Requirements:

PyAudio

---
PyAudio and python3+ is unsupported by default with ```pip install pyaudio```.

So need to run next:

```sudo apt-get install portaudio19-dev python-pyaudio```

and then:

```pip install PyAudio```
